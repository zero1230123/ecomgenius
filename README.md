# ⚡ EcomGenius — AI-Powered E-Commerce Content Engine

Generate product descriptions, ad copy, SEO content, email sequences, brand names, and social media content in seconds.

## 🚀 Features

- **📝 Product Descriptions** — SEO-optimized, conversion-focused
- **📢 Ad Copy Generator** — Facebook, Google, TikTok, Instagram
- **🔍 SEO Optimizer** — Meta titles, descriptions, keywords
- **📧 Email Sequences** — Welcome, abandoned cart, promo
- **💡 Brand Name Generator** — Creative names with taglines
- **📱 Social Media Content** — Captions, hashtags, viral ideas

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript (no frameworks)
- **AI:** OpenRouter API (GPT-4o, Claude 3.5, Llama 3)
- **Hosting:** Vercel (free tier)
- **Design:** Custom dark-mode UI

## ⚡ Quick Start

1. Clone this repo
2. Deploy to Vercel
3. Open the app → Settings → Add your OpenRouter API key
4. Start generating!
